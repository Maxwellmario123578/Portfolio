### 🎯 Comment Tester

1. **Lancer le serveur de développement** :
```bash
cd Portfolio
npm start
```

2. **Ouvrir dans le navigateur** :
```
http://localhost:4200
```

3. **Naviguer vers la section Compétences** :
   - Cliquez sur "Compétences" dans le menu
   - Testez chaque catégorie (Langages, Frameworks, Data Science, Outils)

4. **Tester les interactions** :
   - Survolez une compétence pour voir l'icône et le niveau
   - Observez la barre de progression animée
   - Vérifiez les pourcentages
